    为什么要写这篇文章？

    假设我们正在开发和维护一个 Android 项目，名叫：MyProject。

    一、接入步骤

    （一）安装 Magpie 脚手架（Install cli）

    pub global activate magpie_cli

    （二）Start Workflow

    1.Create module project

    mgpcli create -n flutter_sample

    2.Start service

    cd flutter_sample
mgpcli start

    运行了mgpcli start命令会自动打开 magpie 的可视化管理页面

    3.Switch to browser
    The browser should have been opened instantly，http://127.0.0.1:8080
