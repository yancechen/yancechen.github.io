Magpie 接入流程详解

· 2020-06-09 · # Flutter

本篇文章会详细介绍 magpie 的接入流程

开源项目名称：magpie

github地址：
https://github.com/wuba/magpie
https://github.com/wuba/magpie_sdk
https://github.com/wuba/magpie_fly
https://github.com/wuba/magpie_log

关于 magpie 的介绍：
Magpie：平台工具链开发实践
Magpie：混合开发工程化框架
Magpie：Magpie在安居客有料业务的落地实践

为什么要写这篇文章？

假设我们正在开发和维护一个 Android 项目，名叫：MyProject。

一、接入步骤

（一）安装 Magpie 脚手架（Install cli）

pub global activate magpie_cli

（二）Start Workflow

1.Create module project

mgpcli create -n flutter_sample

2.Start service

cd flutter_sample
mgpcli start

运行了mgpcli start命令会自动打开 magpie 的可视化管理页面

3.Switch to browser
The browser should have been opened instantly，http://127.0.0.1:8080
